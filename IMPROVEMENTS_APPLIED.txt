✅ FIX #1: REMOVED DOUBLE SMOOTHING (Label Smoothing Conflict)
─────────────────────────────────────────────────────────────

WHAT WAS CHANGED:
  Before: F.cross_entropy(s_logits, labels, label_smoothing=0.1)
  After:  F.cross_entropy(s_logits, labels)

WHY:
  - Soft targets from teacher already provide label smoothing
  - Adding label_smoothing=0.1 on top causes over-smoothing
  - Targets become too soft, learning signal becomes unclear
  - Removing it allows soft targets to do the smoothing

EXPECTED IMPROVEMENT: +0.5-1.5% accuracy

────────────────────────────────────────────────────────────────────────────


✅ FIX #2: CURRICULUM LEARNING WITH SCHEDULED DISTILLATION
──────────────────────────────────────────────────────────

WHAT WAS CHANGED:

  1. Configuration (lines 49-53):
     OLD:
       DISTILL_TEMP = 4.0
       DISTILL_ALPHA = 0.1

     NEW (Scheduled):
       DISTILL_TEMP_START = 8.0   → DISTILL_TEMP_END = 2.0
       DISTILL_ALPHA_START = 0.5  → DISTILL_ALPHA_END = 0.05

  2. Added scheduling function (new lines 263-271):
     get_distill_schedule(epoch, total_epochs)
       - Returns scheduled temp and alpha based on training progress
       - Linear decay from start to end values over 200 epochs

  3. Updated training loop (line 485):
     distill_temp, distill_alpha = get_distill_schedule(epoch, EPOCHS)

  4. Updated loss computation (line 500):
     Uses distill_temp and distill_alpha instead of constants

WHY THIS CURRICULUM WORKS:

  Early epochs (Epoch 1-50):
    TEMP = 8.0 (soft) → high probability smoothing
    ALPHA = 0.5 (balanced) → equal weight to hard and soft targets
    → Heavy reliance on teacher guidance
    → Student learns general patterns

  Middle epochs (Epoch 50-150):
    TEMP = 4.0-5.0 (medium)
    ALPHA = 0.2-0.3 (leaning toward hard)
    → Transitioning to harder targets
    → Student becomes more independent

  Late epochs (Epoch 150-200):
    TEMP = 2.0 (crisp) → very sharp probability distribution
    ALPHA = 0.05 (mostly hard) → 95% hard labels, 5% soft
    → Student refines fine details with hard labels
    → Teacher guidance is minimal

EXPECTED IMPROVEMENT: +1-2% accuracy
